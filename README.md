### ðŸ“ File Descriptions

| File                  | Description                                                                 |
| --------------------- | --------------------------------------------------------------------------- |
| `add.py`              | Adds two integers and prints the result.                                    |
| `fizzbuzz.py`         | Classic FizzBuzz logic from 1 to 100.                                       |
